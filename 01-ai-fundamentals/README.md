# AI Fundamentals

## Concepts to Learn

- Artificial intelligence: systems that perform tasks requiring reasoning, perception, language, planning, or decision-making.
- Machine learning: systems that learn patterns from data.
- Deep learning: ML using neural networks with many layers.
- Generative AI: models that generate text, code, images, audio, video, or actions.
- LLM: large language model trained to predict and generate language/code.
- Foundation model: general model adapted to many tasks.
- Inference: running a model to produce output.
- Fine-tuning: further training a model on domain/task data.
- RAG: grounding model answers with retrieved external knowledge.

## DevOps/SRE Translation

Think of an AI app as a distributed production service:

- Model provider is an external dependency.
- Prompt is application logic.
- Context is runtime input.
- Retrieval is a data dependency.
- Agent tools are privileged integrations.
- Evals are regression tests.
- Traces and token logs are observability.
- Guardrails and approvals are safety controls.

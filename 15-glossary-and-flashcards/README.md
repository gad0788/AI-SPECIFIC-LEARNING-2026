# Glossary and Flashcards

This folder is for AI vocabulary and quick review notes.

Use it to build fluency from beginner concepts to advanced production AI terminology.

## Files

- [glossary.md](glossary.md): definitions of core AI, LLM, RAG, MCP, agent, MLOps, and LLMOps terms.

## How to Use

- Add every unfamiliar term you encounter.
- Write definitions in your own words.
- Add one practical DevOps/SRE example for important concepts.
- Convert hard concepts into flashcards.

## Example Flashcard Format

Question:

- What is RAG?

Answer:

- Retrieval-Augmented Generation is a pattern where an LLM answers using external retrieved context, usually from documents indexed in a vector database.

# Glossary

- Agent: AI system that can use tools, observe results, and continue toward a goal.
- AI: broad field of systems that perform tasks associated with intelligence.
- AIOps: applying AI to IT operations workflows.
- Chunking: splitting documents into retrieval-sized pieces.
- Context window: amount of text/tokens a model can use in one request.
- Embedding: numerical representation of text/image/etc. used for similarity search.
- Eval: test that measures model or application behavior.
- Fine-tuning: additional training of a model on task/domain data.
- Function calling: model producing structured arguments for a tool/function.
- Guardrail: control that restricts unsafe or incorrect AI behavior.
- Hallucination: unsupported or false model output.
- LLM: large language model.
- LLMOps: operational practices for LLM applications.
- MCP: Model Context Protocol, a standard for connecting AI clients to tools and data.
- MLOps: operational practices for ML systems.
- Prompt: instructions and context sent to a model.
- Prompt injection: malicious or accidental input that tries to override instructions.
- RAG: retrieval-augmented generation.
- Reranker: model/system that reorders retrieved results by relevance.
- Tool: external function/API/action an agent can call.
- Vector database: database optimized for similarity search over embeddings.

# AI-Specific Learning 2026

Purpose: build a practical AI engineering skill stack on top of DevOps, SRE, and cloud engineering experience.

This workspace is designed to track learning from beginner to advanced:

- AI fundamentals, LLMs, prompt engineering, embeddings, RAG, vector databases.
- AI agents, MCP, tool use, multi-agent systems, memory, permissions, and guardrails.
- Coding assistants: Codex, Claude Code, GitHub Copilot, Cursor-style workflows.
- MLOps, LLMOps, AIOps, observability, evaluation, security, cost, and reliability.
- Cloud AI platforms: AWS Bedrock and AgentCore, Azure AI Foundry, Google Vertex/Gemini, NVIDIA NeMo.
- Hands-on labs and resume-ready proof points for AI, DevOps, SRE, and cloud roles.

Start here:

1. Read [00-start-here/START-HERE.md](00-start-here/START-HERE.md).
2. Follow [00-start-here/ROADMAP.md](00-start-here/ROADMAP.md).

4. Convert completed work into resume bullets in [14-resume-and-ats/resume-skill-bank.md](14-resume-and-ats/resume-skill-bank.md).

Working rule: every concept should eventually become one of these artifacts:

- A note explaining it in your own words.
- A lab or small project.
- A diagram or architecture decision record.
- A resume bullet with measurable impact.
- A GitHub repo or demo you can discuss in interviews.

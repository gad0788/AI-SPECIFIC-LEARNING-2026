# OpenAI Agents

Study areas:

- Responses API for stateful, tool-using model interactions.
- Agents SDK for agent workflows, handoffs, tracing, and tools.
- Codex as a coding agent for repo tasks, reviews, debugging, and MCP integrations.
- Apps SDK and MCP for connecting tools and UI surfaces.

Key resources:

- OpenAI API docs: https://developers.openai.com/api/
- Responses API: https://developers.openai.com/api/reference/responses/overview
- Agents SDK: https://openai.github.io/openai-agents-python/
- Agents SDK GitHub: https://github.com/openai/openai-agents-python
- Codex docs: https://developers.openai.com/codex/

Hands-on:

- Build a simple tool-calling agent.
- Add tracing and eval logs.
- Build a DevOps helper tool: read a YAML file, detect risk, return recommendations.

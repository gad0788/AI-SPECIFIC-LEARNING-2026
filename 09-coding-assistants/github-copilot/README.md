# GitHub Copilot Notes

Study:

- Copilot Chat.
- Agent mode.
- Pull request assistance.
- GitHub Actions troubleshooting.
- MCP support in VS Code where available.

Practice:

- Ask Copilot to explain a GitHub Actions workflow.
- Use it to draft a CI fix.
- Compare output quality with Codex and Claude Code.

Security:

- Do not paste secrets.
- Review suggested code.
- Run tests.
- Keep generated code consistent with repo standards.

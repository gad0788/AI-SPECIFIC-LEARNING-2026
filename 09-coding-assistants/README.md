# Coding Assistants

Coding assistants are AI agents for software engineering work. Treat them as junior-to-mid engineers that can move fast but need review.

## Tools to Compare

- OpenAI Codex
- Anthropic Claude Code
- GitHub Copilot
- Cursor/Windsurf-style IDE agents

## Skills to Build

- Writing precise tasks.
- Giving repo context.
- Asking for plans before large changes.
- Reviewing diffs.
- Running tests.
- Using agents for code review.
- Using MCP safely.
- Keeping secrets out of prompts.
- Creating reusable project instructions.

## DevOps/SRE Use Cases

- Terraform module review.
- Kubernetes manifest validation.
- CI/CD failure debugging.
- Log parsing scripts.
- Runbook generation.
- Incident timeline extraction.
- Cloud cost script generation.

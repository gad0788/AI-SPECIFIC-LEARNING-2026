# Inbox

This folder is for raw capture before organizing.

Use it when you find a link, idea, project, concept, or resume bullet but do not yet know where it belongs.

## Files

- [capture.md](capture.md): scratchpad for links, concepts, project ideas, and resume bullet ideas.

## Cleanup Routine

Once a week:

- Move resource links to `12-github-repos-and-resources`.
- Move definitions to `15-glossary-and-flashcards`.
- Move project ideas to `10-hands-on-labs`.
- Move resume ideas to `14-resume-and-ats`.
- Delete or archive anything no longer useful.

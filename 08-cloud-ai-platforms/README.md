# Cloud AI Platforms

Cloud AI platforms matter because enterprises want AI systems that integrate with identity, networking, logging, compliance, data, and CI/CD.

## AWS

- Amazon Bedrock
- Bedrock Agents
- Bedrock AgentCore
- Bedrock Knowledge Bases
- Lambda/ECS/EKS for tools and apps
- CloudWatch for logs/metrics
- IAM for least privilege

## Azure

- Azure AI Foundry
- Azure AI Foundry Agent Service
- Azure OpenAI
- Azure AI Search
- Azure Monitor
- Key Vault

## Google Cloud

- Vertex AI / Gemini platform
- Vertex AI Agent Builder
- Agent Development Kit
- Cloud Run
- BigQuery
- Cloud Logging

## NVIDIA

- NIM for model serving.
- NeMo Guardrails for safety and control.
- NeMo ecosystem for enterprise AI workflows.

## Study Method

For each cloud:

1. Build one small app.
2. Add auth/secrets.
3. Add logs/metrics/traces.
4. Add cost controls.
5. Add CI/CD.
6. Write a production readiness checklist.

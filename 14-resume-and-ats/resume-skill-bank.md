# Resume and ATS Skill Bank

Do not add a skill to your resume just because you read about it. Add it when you can explain it and show evidence.

## AI Skills to Earn

Beginner:

- LLM API integration
- Prompt engineering
- Structured output
- Embeddings
- Basic RAG
- Vector database basics
- AI coding assistants

Intermediate:

- Agent tool calling
- MCP client/server configuration
- RAG evaluation
- LLM observability

Advanced:

- Multi-agent orchestration

- Model routing and fallback architecture

## Resume Bullet Formula

Use this structure:

`Built/Deployed/Automated [AI system] using [tools] to achieve [measurable outcome], with [reliability/security/observability detail].`

Examples to customize only after completing real work:

- Built a RAG assistant over DevOps runbooks using FastAPI, embeddings, and a vector database, adding citation-backed responses, retrieval evaluation, and request tracing.
- Deployed an AI agent workflow on AWS using Bedrock, Lambda/ECS, IAM-scoped tools, CloudWatch logs, and human approval gates for production-safe automation.
- Implemented LLM observability with OpenTelemetry-style traces for model calls, retrieval latency, token usage, tool invocations, and error rates.
- Configured MCP integrations for GitHub, GitLab, Jira/Confluence, and Linear to support AI-assisted DevOps workflows with controlled authentication.
- Designed an LLMOps pipeline with prompt versioning, eval datasets, CI regression checks, model fallback, and cost monitoring.

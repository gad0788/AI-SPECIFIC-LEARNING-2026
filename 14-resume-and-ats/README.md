# Resume and ATS

This folder converts AI learning into resume-ready evidence.

The goal is to add AI engineering skills on top of DevOps, SRE, and cloud engineering without making unsupported claims.

## Files

- [resume-skill-bank.md](resume-skill-bank.md): AI skills to earn and resume bullet examples.
- [ats-keywords.md](ats-keywords.md): keywords for AI, LLMOps, DevOps, SRE, and cloud roles.
- [project-evidence-tracker.md](project-evidence-tracker.md): table for tracking portfolio-ready projects.

## Rule

Only add a skill to your resume when you can show proof:

- Code.
- Demo.
- Architecture diagram.
- Eval results.
- Observability dashboard.
- Deployment.
- Written explanation.

## Target Resume Positioning

Cloud/DevOps/SRE engineer growing into AI platform, LLMOps, agentic AI, and production AI engineering roles.

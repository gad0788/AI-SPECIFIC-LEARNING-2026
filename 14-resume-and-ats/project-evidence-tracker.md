## Evidence Quality Bar

A project is resume-ready when it has:

- [ ] Working code.
- [ ] README with setup and demo.
- [ ] Architecture diagram.
- [ ] Tests or evals.
- [ ] Security notes.
- [ ] Observability notes.
- [ ] Measurable outcome or comparison.

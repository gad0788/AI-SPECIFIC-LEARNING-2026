# Vector Databases

Vector databases store embeddings and support similarity search.

## Options to Learn

- Chroma: simple local prototyping.
- pgvector: Postgres extension, good for teams already using Postgres.
- Qdrant: open-source vector database with strong filtering and performance.
- Weaviate: open-source AI-native vector database with modules and cloud option.
- Milvus: open-source, scalable vector database.
- Pinecone: managed vector database.
- Elasticsearch/OpenSearch: hybrid search and vector search in existing search stacks.

## Comparison Criteria

- Local development experience.
- Managed cloud option.
- Metadata filtering.
- Hybrid search support.
- Multi-tenancy and access control.
- Backup/restore.
- Observability.
- Cost.
- Operational complexity.
